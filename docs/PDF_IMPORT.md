# Import PDF ŽoP

PyMuPDF získává text po stranách, pdfplumber tabulky SD2. Hlavička a finanční souhrn se
čtou z obsahu dokumentu. Počáteční záloha je cash-flow, nikoli čerpání rozpočtu. Každý
řádek nese číslo stránky. PDF bez dostatečné textové vrstvy končí českou chybou požadující
OCR. Hash a dvojice číslo/verze brání duplicitám.
